# To-Do List

## Description

A GUI-based To-Do List application developed using Python and Tkinter. It helps users manage daily tasks through a simple and interactive interface.

## Features

- Add Tasks
- Delete Tasks
- Mark Tasks as Completed
- Clear All Tasks
- Scrollable Task List
- User-Friendly GUI

## Technologies Used

- Python
- Tkinter
