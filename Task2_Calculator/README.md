# Simple Calculator

## Description

A simple GUI-based calculator developed using Python and Tkinter. It performs basic arithmetic operations through an easy-to-use graphical interface.

## Features

- Addition
- Subtraction
- Multiplication
- Division
- Input Validation
- Clear Function
- User-Friendly GUI

## Technologies Used

- Python
- Tkinter

## Project Structure

```
Task2_Calculator/
│── main.py
│── README.md
│── requirements.txt
```

## How to Run

1. Make sure Python is installed on your system.
2. Open the project folder.
3. Run the following command:

```bash
python main.py
```
