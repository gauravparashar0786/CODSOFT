# Password Generator

## Description

A GUI-based Password Generator developed using Python and Tkinter. It generates secure random passwords based on user-selected criteria.

## Features

- Generate Random Passwords
- Adjustable Password Length
- Uppercase Letters
- Lowercase Letters
- Numbers
- Symbols
- Copy Password to Clipboard
- Clear Button
- User-Friendly GUI

## Technologies Used

- Python
- Tkinter
